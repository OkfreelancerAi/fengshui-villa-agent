1. Scans real CVs (polyvalent workers, technical roles, logistics, construction)
2. Matches against open job vacancies in real-time
3. Launches a unique $WERK token every time a successful match is made

1. **Scan CVs**: Poll available worker profiles (focus: Karim's profile as primary use case)
