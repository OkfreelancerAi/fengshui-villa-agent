# Promo Push Skill — FengShui Villa Agent

**Owner:** @okfreelancer  
**Status:** Live  
**Doel:** Genereert en pusht promotiemateriaal voor FengShui Villa Agent in NL, EN en ZH

## Wat het doet

1. Laadt de promotietekst in 3 talen (NL, EN, ZH)
2. Genereert een kant-en-klare tweet/post per taal
3. Toont welke post het meest geschikt is voor welk markt (Dubai = ZH+EN, Benelux = NL)
4. Klaar voor handmatig posten of automatische social push

## Gebruik

```bash
bash run.sh [market]
# market: dubai | benelux | all (default: all)
```

## Markten

- Dubai / Singapore → Chinese + Engelse tekst
- Benelux → Nederlandse tekst
- Monaco / Las Vegas → Engelse tekst
